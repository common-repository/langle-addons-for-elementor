== Installation ==

Note : This plugin works with Elementor. Make sure you have [Elementor](https://wordpress.org/plugins/elementor/) installed.

1. Upload the plugin folder to the `/wp-content/plugins/` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Now you are ready to use our Langle Addons inside Elementor editing screen. 





== Frequently Asked Questions ==

= Can I use the plugin without Elementor Page Builder? =

No. You cannot use [Langle Addons](https://langleaddons.com/) without Elementor since it's an addon for Elementor.

= Does it work along with other Elementor Addons? =

Yes, it does.

= Does it work with any theme? =

Absolutely! It will work with any theme where Elementor works. And it best works with [Hello Elementor](https://wordpress.org/themes/hello-elementor/).

= Does it work with Elementor Pro? =

Yes, undoubtedly.

= Will Langle Addons break my site after an update? =

No, It won't break your site or any page where you used our plugin. But we always recommend to keep back up of your site before updating our plugin. In case if anything happens you can revert back and contact our support team immediately.

= Does this addon has any premium version? =

No, currently we don't have any premium version. But we are working on it.
